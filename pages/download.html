How to install the game?

Once you have downloaded the game the next step is to install so you can play.

Open the SapphireConquer.exe and use the Browse button to select where you want to install the game.

When you have choosen your destination folder the next step is to click Extract, so all the files will be extracted in your folder.

When the process is finished, click done.

The next step is to navigate to your folder where you have installed the game and open SapphireCO.exe to launch the game.

Once the launcher is opened, you can select on how many FPS you want to play the game and what resolution the game should be started.

Click Play now! and enjoy the game.
